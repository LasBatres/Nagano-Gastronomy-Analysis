# Nagano-Gastronomy-Analysis
Active project where I investigate the relationship between Nagano's restaurants prices and ratings.

This idea begins with my chef-wife desire to live a full-grastronomic japan experience combined with JDM preferences, so Nagano is where both interests converge.
Because of funding and transportation limitations I started to think....
There exist the perfect path where we can experience the balance between prices and flavors on each restaurant?
Once I localize the most convenient restaurants for each type of cuisin how can a go there?

-------------------------------------

## TECHNICAL GOALS
1. Create the Pipeline using Tabelog as basis.
2. Save all usefull and available information for the purpose intended for each restauran.
3. Clear all data for machine and preson-reading porpuses.

## ANALYSIS GOALS
1. Determine relationship between prices and rating.
2. Mesure distance for each restaurtan to it's closer public transport station.
3. Find the satisfaction function tu evaluate each restaurant.
4. Optimize the satisfaction function to fin the best places.
